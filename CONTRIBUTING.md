# Contributing to Python Training Materials

Thank you for your interest in contributing! These materials are used in real training sessions, so quality matters.

## How to Contribute

### Report Issues
- Found a typo or error? Open an issue
- Exercise solution incorrect? Let us know
- Outdated content? Flag it

### Suggest Improvements
- Better explanations for difficult concepts
- Additional exercises
- New topics or modules

### Submit Changes
1. Fork this repository
2. Create a branch: `git checkout -b fix/description`
3. Make your changes
4. Test any code examples
5. Submit a PR with a clear description

## Guidelines

- **Keep it practical** — Focus on exercises and real-world examples
- **Test your code** — All code examples should run without errors on Python 3.10+
- **Be clear** — These materials are for learners, so clarity beats cleverness
- **Include solutions** — Every exercise should have a solution in `exercises/solutions/`

## Style

- Use Python 3.10+ features and syntax
- Follow PEP 8
- Include type hints in examples (intermediate+)
- Use docstrings in function examples
